Qt5 リポジトリのミラーはこちらで行なっています。

リポジトリをクローンする際の例:

$ git clone git://git.qt-users.jp/qt/qt5.git

又は

$ git clone http://git.qt-users.jp/qt/qt5.git

Qt は GPL v3, LGPL v2 そして 商用版ライセンスのもとで頒布されています。ライセンスについて更に知りたい 場合は ここを見て下さい。

Qt 商用版 ライセンス
Qt 商用版ライセンスはサポート、アップデート などを含む様々なライセンス条件を提供しています。 Qt 商用版ライセンスは競争上の優位性を確保し、 開発投資を保護するためや、プロプライエタリ又は商用に Qt でソフトウェアを 開発する際の適切なライセンスです。

  • Qt 商用版ライセンスを評価する (qt.digia.com)

MinGW バイナリパッケージは 32 ビット MinGW-builds (gcc 4.8.0, sjlj exception handing, posix threading) toolchain を含んでいます。 ソースは MinGW-builds プロジェクト または、 releases.qt-project.org で利用可能です。

旧バージョン

Qt 4.7 やその他の Qt の旧バージョンは アーカイブ で入手可能です。

Qt がシステムのプラットフォームに対応されているかを確認し、 Qt Documentation にあるインストールの注意を読んでください。

もしスタンドアロンのインストーラーが欲しい場合は、下のリストからOSを参考にして、最 新の Qt Creator を入手してください

  • Qt Creator 3.0.0 - Windows (53 MB)
  • Qt Creator 3.0.0 - Linux/X11 32 ビット (62 MB)
  • Qt Creator 3.0.0 - Linux/X11 64 ビット (61 MB)
  • Qt Creator 3.0.0 - Mac (Intel 32/64 ビット) (53 MB)

Qt Creatorのソースコードは zip (29 MB) または tar.gz (23 MB)として入手可能です。 qt.gitorious.org/qt-creator のリポジトリにもアクセスしてみてください。
