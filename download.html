Qt 商用版 ライセンス
Qt 商用版ライセンスはサポート、アップデート などを含む様々なライセンス条件を提供しています。 Qt 商用版ライセンスは競争上の優位性を確保し、 開発投資を保護するためや、プロプライエタリ又は商用に Qt でソフトウェアを 開発する際の適切なライセンスです。

  • Qt 商用版ライセンスを評価する (qt.digia.com)

Qt 5.1

下のリストからOSを参考にして、最新の Qt 5.1を入手してください。 以下のパッケージは Qt 5.1.0 ライブラリと Qt Creator 2.8.2 を含んでいます。

  • Qt 5.1.0 - Linux 32 ビット (395 MB)
    (代替ダウンロードリンク | 情報)
  • Qt 5.1.0 - Linux 64 ビット (394 MB)
    (代替ダウンロードリンク | 情報)
  • Qt 5.1.0 - Mac (404 MB)
    (代替ダウンロードリンク | 情報)
  • Qt 5.1.0 - Windows 32 ビット (MinGW 4.7, 650 MB)
    (代替ダウンロードリンク | 情報)
  • Qt 5.1.0 - Windows 32 ビット (VS 2010, 485 MB)
    (代替ダウンロードリンク | 情報)
  • Qt 5.1.0 - Windows 32 ビット (VS 2010, OpenGL, 476 MB)
    (代替ダウンロードリンク | 情報)
  • Qt 5.1.0 - Windows 64 ビット (VS 2012, 5.1 MB)
    (代替ダウンロードリンク | 情報)

ここ ( 代替ダウンロードリンク)から分割されたソースパッケージを入手できます。 zip で一つにまとめられたソースパッケージも利用可能です、( Windows 向け 271 MB ) ( 代替ダウンロードリンク | 情報 ) tar.qz として同じように利用可能です。 ( Linux/Mac 向け 220 MB ) (代替ダウンロードリンク|情報 ) 全インストーラーは ここ ( 代替ダウンロードリンク ) で利用可能です。 またはリポジトリ qt.gitorious.org/qt でも利用可能です。

MinGW バイナリパッケージは 32 ビット MinGW-builds (gcc 4.7.2, sjlj exception handing, posix threading) toolchain を含んでいます。 ソースは MinGW-builds プロジェクト または、 releases.qt-project.org で利用可能です。

Qt 4.8

下のリストからあなたのOSを参考にして、最新の Qt 4.8を入手してください。

  • Qt ライブラリ 4.8.5 - Windows (minGW 4.4, 317 MB)
  • Qt ライブラリ 4.8.5 - Windows (VS 2008, 233 MB)
  • Qt ライブラリ 4.8.5 - Windows (VS 2010, 234 MB)
  • Qt ライブラリ 4.8.5 - Linux/X11 (225 MB)
  • Qt ライブラリ 4.8.5 - Mac (185 MB) , debug ライブラリ (480 MB)
  • Qt ライブラリ 4.8.5 - embedded Linux (236 MB)
  • Qt ライブラリ 4.8.5 - Windows CE (269 MB)

Qt 4.8 のソースコードは zip (269 MB) または tar.gz (225 MB)として入手可能です qt.gitorious.org/qt のリポジトリにもアクセスしてみてください。

旧バージョン

Qt 4.7 やその他の Qt の旧バージョンは アーカイブ で入手可能です。

Qt がシステムのプラットフォームに対応されているかを確認し、 Qt Documentation にあるインストールの注意を読んでください。

もしスタンドアロンのインストーラーが欲しい場合は、下のリストからOSを参考にして、最新の Qt Creator を入手してください

  • Qt Creator 2.8.0 - Windows (52 MB)
  • Qt Creator 2.8.0 - Linux/X11 32 ビット (62 MB)
  • Qt Creator 2.8.0 - Linux/X11 64 ビット (61 MB)
  • Qt Creator 2.8.0 - Mac (Intel 32/64 ビット) (53 MB)

Qt Creatorのソースコードは zip (27 MB) または tar.gz (21 MB)として入手可能です。 qt.gitorious.org/qt-creator のリポジトリにもアクセスしてみてください。
