Qt5 リポジトリのミラーはこちらで行なっています。

リポジトリをクローンする際の例:

$ git clone git://git.qt-users.jp/qt/qt5.git

又は

$ git clone http://git.qt-users.jp/qt/qt5.git

Qt は GPL v3, LGPL v2 そして 商用版ライセンスのもとで頒布されています。ライセンスについて更に知りたい 場合は ここを見て下さい。

Qt 商用版 ライセンス
Qt 商用版ライセンスはサポート、アップデート などを含む様々なライセンス条件を提供しています。 Qt 商用版ライセンスは競争上の優位性を確保し、 開発投資を保護するためや、プロプライエタリ又は商用に Qt でソフトウェアを 開発する際の適切なライセンスです。

  • Qt 商用版ライセンスを評価する (qt.digia.com)

ここ から分割されたソースパッケージを入手できます。 zip で一つにまとめられたソースパッケージも利用可能です、( Windows 向け 282 MB ) ( 情報 ) tar.qz として同じように利用可能です。 ( Linux/Mac 向け 227 MB ) ( 情報 ) 全インストーラーは ここ で利用可能です。 またはリポジトリ qt.gitorious.org/qt でも利用可能です。

MinGW バイナリパッケージは 32 ビット MinGW-builds (gcc 4.8.2, sjlj exception handing, posix threading) toolchain を含んでいます。 ソースは MinGW-builds プロジェクト または、 download.qt-project.org で利用可能です。

Windows オフラインインストーラーはデフォルトで ANGLE に基づいています。Windows での ANGLE パッケージ と OpenGL パッケージの違いについて知りたい場合は、 Qt 5 on Windows ANGLE and OpenGL にアクセスして下さい。

旧バージョン

Qt 5.2、Qt 4.7 やその他の Qt の旧バージョンは アーカイブ で入手可能です。

Qt がシステムのプラットフォームに対応されているかを確認し、 Qt Documentation にあるインストールの注意を読んでください。

Qt Creator 3.1.2 は、Qt 5.3.1 パッケージに含まれていますが、 もしスタンドアロンのインストーラーが欲しい場合は、下のリストからOSを参考にして、最 新の Qt Creator を入手してください

  • Qt Creator 3.1.2 - Windows (69 MB) (情報)
  • Qt Creator 3.1.2 - Linux/X11 32-bit (84 MB) (情報)
  • Qt Creator 3.1.2 - Linux/X11 64-bit (82 MB) (情報)
  • Qt Creator 3.1.2 - Mac (68 MB) (情報)

Qt Creatorのソースコードは zip (28 MB) または tar.gz (21 MB)として入手可能です。 qt.gitorious.org/qt-creator のリポジトリにもアクセスしてみてください。
